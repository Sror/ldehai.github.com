How to Migrate to iOS7

The new version iOS7 changed the statusbar, now default, the statusbar overlap UIViewController, so if you have items on top of view, it’s will display under statusbar.

The simplest solution is add “View controller-based status bar appearance” to plist, and set it’s value to “NO”.
